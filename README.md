## User Stories
As a User, I want to be able to:
- Create a new book
  - Title
  - Author
  - Genre
  - Total Page Count
  - Book Status
    - Read
    - Currently Reading (pg number)
    - Not Read
- See the book I've created on an index page
- Edit the books I've created
- Delete my book
